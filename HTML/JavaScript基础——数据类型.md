## 字符串string
通过单引号或者双引号创建字符串
```
var string = "字符串"
```
字符串是只读类型，一旦创建不能修改
## 数值number
八进制表示是在数字前面加0，十六进制是在数字前面加0x
##布尔类型
该类型只有两个值true和false
##获取数据类型typeof
```
       console.log(typeof "字符串")
       console.log(typeof 250)
       console.log(typeof true)
       console.log(typeof null)
       console.log(typeof {name:"bx"})
       console.log(typeof function () {})
       console.log(typeof undefined)
```
