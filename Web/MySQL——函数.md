## 系统信息
版本信息
```
SELECT VERSION();
+-----------+
| VERSION() |
+-----------+
| 5.7.21    |
+-----------+
1 row in set (0.00 sec)
```
当亲用户的连接数
```
SELECT CONNECTION_ID();
+-----------------+
| CONNECTION_ID() |
+-----------------+
|               5 |
+-----------------+
```
查询当前用户的连接信息
```
HOW PROCESSLIST;
+----+------+-----------+-------+---------+------+----------+------------------+
| Id | User | Host      | db    | Command | Time | State    | Info             |
+----+------+-----------+-------+---------+------+----------+------------------+
|  5 | root | localhost | jblog | Query   |    0 | starting | SHOW PROCESSLIST |
+----+------+-----------+-------+---------+------+----------+------------------+
```
查看当前连接的数据库
```
SELECT DATABASE();
+------------+
| DATABASE() |
+------------+
| jblog      |
+------------+
```
